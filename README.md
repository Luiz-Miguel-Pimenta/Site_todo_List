# ✅ QuickList — Lista de Tarefas

Uma aplicação web simples e objetiva para gerenciar suas tarefas do dia a dia, desenvolvida com HTML, CSS e JavaScript puro.

## 📋 Sobre o Projeto

O **QuickList** é um site de lista de tarefas (To-Do List) que permite adicionar, marcar como concluída e remover tarefas de forma rápida e intuitiva, sem necessidade de login ou instalação. Toda a lógica da aplicação é gerenciada no front-end com JavaScript.

## ✨ Funcionalidades

- ➕ Adicionar novas tarefas
- ✔️ Marcar tarefas como concluídas com checkbox interativo
- 🗑️ Deletar tarefas individualmente
- 🎨 Interface limpa e responsiva
- 🔤 Tipografia moderna com a fonte **Inter** (Google Fonts)

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página
- **CSS3** — Estilização e layout responsivo
- **JavaScript** — Lógica de criação, conclusão e remoção de tarefas
- **Google Fonts** — Fonte Inter para melhor legibilidade

## 📁 Estrutura do Projeto

```
Site_todo_List/
├── img/            # Imagens utilizadas no projeto
├── index.html      # Estrutura principal da aplicação
├── styles.css      # Estilos e layout
└── script.js       # Lógica de gerenciamento das tarefas
```

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Luiz-Miguel-Pimenta/Site_todo_List.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Site_todo_List
   ```

3. Abra o arquivo `index.html` no seu navegador — não é necessário nenhum servidor ou instalação adicional.

## 🖥️ Como Utilizar a Interface

1. Digite o nome da tarefa no campo **"Adicione uma nova tarefa"**
2. Clique em **Adicionar** ou pressione Enter
3. A tarefa aparecerá na lista abaixo
4. Clique no **checkbox** para marcar a tarefa como concluída
5. Clique no **ícone de lixeira** para remover a tarefa da lista
